# random-dataset-generator

[English](./README.md) | 中文说明

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个轻量、灵活的**随机合成数据集生成器**，用于测试、原型开发和机器学习实验。

## ✨ 特性

- 生成随机表格数据（CSV、JSON、Excel）
- 支持多种数据类型：整数、浮点数、字符串、日期、布尔值、类别
- 可自定义列名、数据分布、缺失值比例
- 定义约束（最小/最大值、候选项、正则表达式）
- 任意大小的数据集（行 × 列）
- 提供代码 API 和命令行工具（CLI）
