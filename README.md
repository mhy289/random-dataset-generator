# random-dataset-generator

[中文说明](./README_zh.md) | English

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A lightweight and flexible tool to generate **synthetic random datasets** for testing, prototyping, and machine learning experiments.

## ✨ Features

- Generate random tabular data (CSV, JSON, Excel)
- Support multiple data types: integer, float, string, date, boolean, categorical
- Customize column names, distributions, missing value ratio
- Define constraints (min/max, choice lists, regex patterns)
- Generate dataset of any size (rows × columns)
- Code-first API + command line interface (CLI)
